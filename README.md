# Spotify New Releases

Website built to show the latest releases of your followed artists on Spotify. Users can filter by artist name or by date released.

## Usage

Clone oath-bridge-template as well. Export spotify credentials and redirect URI in the command line.

## Install Dependencies

```
npm install
```

## Run App

```
npm run start
```
